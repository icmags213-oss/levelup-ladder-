📲 Install LevelUp on your phone

Install LevelUp as an app on your home screen so it works offline and feels like a native app.

☁️ Cloud sync (for co-coordinators)

Share ladder data between coordinators using your GitHub repo. After a session, generate data.json and upload it to GitHub. Other coordinators can then pull the latest data.

Workflow: 1) Tap "Generate data.json" → file downloads. 2) Upload it to your GitHub repo (replace existing data.json). 3) Other coordinator taps "Load from cloud" before their next session.

⚠️ Note: "Load from cloud" replaces your current data. Make a local backup first if unsure.

📤 Publish public standings

Generate a standalone standings.html file you can upload to GitHub Pages. Players can view the ladders and their own stats — read-only.

After downloading, upload it to your GitHub repo (same place as index.html) so players can visit your-site/standings.html.
